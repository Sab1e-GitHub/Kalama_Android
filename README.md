# Kalama_Android

Kalama 热成像安卓端上位机

热成像传感器采用的是我祖传的 AMG8833，8 * 8的像素实在是难以言表，于是我添加了相机辅助模式（就是预览相机），效果还是很不错的。
8 * 8的像素双线性插值插到72 * 72，效果还可以，帧数表现也不错，再使用插帧后可以说相当丝滑了。
